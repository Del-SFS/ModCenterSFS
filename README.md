# SCAN - ModCenterSFS

This is the official **ModBase** for the **SCAN** Mod Manager.

### 🚀 Purpose
This repository is a centralized hub designed to:
* **Host** Spaceflight Simulator (SFS) mods.
* **Provide** automated `mod.json` metadata for the **SCAN** engine.
* **Streamline** the installation process for the **SFSMI** (Spaceflight Simulator Mod Installer).

### 📂 Structure
Each mod is organized into its own folder containing:
1. `mod.json` - Metadata (version, author, type).
2. `ModName.zip` - The actual mod files for the game.
